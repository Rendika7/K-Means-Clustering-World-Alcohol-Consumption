### Tentang Dataset

Dataset ini menyediakan informasi tentang konsumsi alkohol berdasarkan negara, termasuk jenis alkohol yang dikonsumsi, jumlah konsumsi per kapita, dan lebih banyak lagi. Anda dapat mengakses datasetnya [di sini](https://www.kaggle.com/datasets/heemalichaudhari/world-alcohol-consumption-dataset/data).

---

### Tujuan

- **K-Means Clustering**: Terapkan algoritma K-Means untuk mengelompokkan negara-negara berdasarkan pola konsumsi alkohol mereka.
- **Insight Discovery**: Temukan kelompok unik atau tren menarik dalam pola konsumsi alkohol di seluruh dunia.
- **Visualisasi**: Visualisasikan kelompok-kelompok yang dihasilkan untuk pemahaman yang lebih baik.
